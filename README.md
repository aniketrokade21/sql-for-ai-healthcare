# sql-for-ai-healthcare
Practicing data extraction and cleaning for AI model preparation.
📄 Healthcare Data Analysis with SQL
Project Goal: Mastering data extraction and transformation using SQL, specifically focusing on complex healthcare datasets to prepare for AI model integration.

🛠️ Technical Skills Demonstrated
Data Cleaning: Standardizing patient records and handling missing values.

Relational Mapping: Joining Patients, Visits, and Provider tables to create unified views.

Aggregation: Calculating hospital KPIs (e.g., average length of stay, patient demographics).

Filtering: Using advanced WHERE logic to segment high-risk patient groups.

📁 Repository Structure
/scripts/01_basic_selection.sql - Fundamental queries and filtering.

/scripts/02_complex_joins.sql - Relational data structures.

/scripts/03_data_cleaning.sql - Handling NULLs and formatting issues.

💡 The "AI Connection"
As a transition into the AI space, these scripts demonstrate the Pre-processing Phase. Before an AI can predict patient outcomes, the data must be queried, cleaned, and structured—which is what this repository document.
